聊天盒 (Chatbox)

基于 Node.js 与 Socket.io 的聊天盒应用，外观简洁，易于管理。

Download as .zip Download as .tar.gz View on GitHub

该聊天盒基于Node.js与Socket.io，充分利用了HTML5 Websocket双向通讯技术，在方便网站游客高速实时聊天的同时也提供网站管理员强大的控制面板，管理员可对全体游客或者特定游客进行各种操作。

安装并启动聊天盒

$ cd chatbox
$ npm install
$ node index.js

内嵌聊天盒

你只需在你的网页中引入下面的几个文件即可：

<link rel="stylesheet" href="http://你的域名.com:端口/client.css">
<script src="https://code.jquery.com/jquery-1.9.0.min.js"></script>
<script src="https://cdn.socket.io/socket.io-1.4.5.js"></script>
<script src="http://你的域名.com:端口/client.js"></script>

托管聊天盒

无需安装，直接复制粘贴下面代码即可：

<link rel="stylesheet" href="http://lifeislikeaboat.com:2007/client.css">
<script src="https://code.jquery.com/jquery-1.9.0.min.js"></script>
<script src="https://cdn.socket.io/socket.io-1.4.5.js"></script>
<script src="http://lifeislikeaboat.com:2007/client.js"></script>
